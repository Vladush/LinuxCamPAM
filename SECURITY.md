# Security Policy

## Supported Versions

| Version | Supported                               |
| ------- | --------------------------------------- |
| 0.9.x   | :white_check_mark: (active development) |
| < 0.9   | :x:                                     |

## Reporting a Vulnerability

**Do not open a public issue.**

If you discover a security vulnerability in this project, please email [contact.github.submerge594@slmails.com](mailto:contact.github.submerge594@slmails.com). We will get back to you within 48 hours.

## Threat Model

We take security seriously. Please refer to our detailed [Security Assessment](docs/SECURITY_ASSESSMENT.md) for an overview of the threat model, protected assets, and design decisions regarding the daemon and standard users.
